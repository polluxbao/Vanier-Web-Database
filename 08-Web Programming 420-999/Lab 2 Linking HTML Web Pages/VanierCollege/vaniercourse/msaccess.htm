MS Access Course

Microsoft Access allows me to design and create a simple database for insurance data management. What is the difference between Exccel worksheet and Access? The main difference is organizing data into related tables for example:

  1. Client table for storing insurance client information.
  2. insurance_products table for storing products offered by insurance company.
  3. Enrollment_client table for storing the payment made by clients.
